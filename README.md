# L'Essence Parfumerie

Website für eine fiktive Luxus-Parfum-Marke, umgesetzt im Modul M293 mit reinem HTML, CSS und JavaScript (ohne Framework).

## Kurzbeschreibung

L'Essence Parfumerie ist eine mehrseitige Website für ein fiktives Parfumhaus. Sie umfasst eine Startseite, eine Kollektionsseite mit filterbarem Produktgrid, zehn einzelne Produktseiten mit Video, eine "House"-Seite (About), sowie eine Kontaktseite mit Formular. Das Design ist dunkel gehalten mit goldenen Akzenten, großen Produktbildern und dezenten Animationen (Scroll-Reveal, Ladeanimation, individueller Cursor).

**Technik:** HTML5, CSS3 (eigenes Stylesheet, responsive mit Media Queries), Vanilla JavaScript (Navigation, Produktfilter, Formularvalidierung, Scroll-Effekte). Die Produktseiten werden über ein Python-Skript (`build_pages.py`) aus einer zentralen Produktliste generiert, damit alle Seiten die gleiche Struktur haben.

## Struktur

```
index.html            Startseite
collection.html        Produktübersicht mit Filter
product-*.html          10 einzelne Produktseiten
about.html              Über das Haus
contact.html             Kontaktformular
css/styles.css           Gesamtes Styling
js/main.js                Interaktivität (Navigation, Filter, Formular, Animationen)
images/, videos/          Medien
docs/                      Styleguide, Wireframes, KI-Einsatz (Kurzfassung)
build_pages.py             Generiert die Produktseiten aus den Produktdaten
```
